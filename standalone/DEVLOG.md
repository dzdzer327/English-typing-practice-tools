## 2026-06-01

### 项目启动

今天完成了英语打字练习项目的初始化和后端搭建。

### 完成事项

**1. 设计文档**
- 完成 `DESIGN.md`，确定技术选型、功能模块、界面方案
- 原计划 Electron 做桌面端，后调整为前后端分离

**2. 后端搭建（Spring Boot）**
- 技术栈：Spring Boot 3.2.5 + JPA + MySQL + Java 17
- 包结构：controller / service / repository / entity / dto / config / exception

**3. 数据库设计**

三张核心表：

| 表 | 说明 |
|----|------|
| `users` | 用户信息、统计数据、打卡连续天数 |
| `practice_records` | 每次练习的 WPM、准确率、时长、错误数 |
| `check_ins` | 每日打卡记录，含当日练习汇总 |

**4. API 接口**

- 认证：注册、登录、获取用户信息
- 练习：提交记录、查询记录、每日统计、总览
- 打卡：每日打卡、打卡统计、打卡日历、历史记录
- 内容：随机单词、随机句子、随机文章、单词查询

**5. 外部 API 接入**
- Random Word API — 随机单词
- Free Dictionary API — 例句和单词释义
- 策略：外部优先，超时 3 秒自动降级到内置词库

### 遇到的问题
- H2 的 `FUNCTION('DATE', ...)` 语法不兼容 MySQL，改用原生 SQL
- 外部 API 网络不稳定，需要内置数据兜底
